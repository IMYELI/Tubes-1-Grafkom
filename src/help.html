How to Use

  1. Draw Line

    • Pilih button Line
    • Klik sekali pada canvas untuk titik pertama
    • Klik kembali untuk membentuk titik kedua sekaligus sebuah garis

  2. Draw Square

    • Pilih button Square
    • Klik sekali pada canvas
    • Hover mouse pada canvas hingga ukurannya sesuai dengan yang diinginkan
    • Klik kembali untuk membentuk memastikan titik yang dipilih dan membentuk sebuah persegi

  3. Draw Rectangle

    • Pilih button Rectangle
    • Klik sekali pada canvas
    • Hover mouse pada canvas hingga ukurannya sesuai dengan yang diinginkan
    • Klik kembali untuk membentuk memastikan titik yang dipilih dan membentuk sebuah persegi panjang

  4. Polygon

    • Pilih button Polygon
    • Klik sekali pada canvas
    • Klik pada canvas sebanyak titik yang diinginkan. Urutan titik dibebaskan karena digunakan algoritma Convex Hull
    • Klik kanan untuk menghentikan pembuatan polygon

How to Interact with Model

  1. Memilih bangun

      • Pilih button Select
      • Klik salah satu bangun ruang yang ada dicanvas
      • Bangun ruang yang dipilih sudah dapat di-interact dengan fungsionalitas lain

  2. Mengubah panjang garis

      With Move Point

      • Pilih button Select
      • Pilih salah satu garis padacanvas
      • Pilih button Move Point
      • Klik pada salah satu titik sudut
      • Hover untuk menyesuaikan panjang garis yang diinginkan
      • Klik pada canvas untuk mengubah posisi titik dan menghasilkan garis dengan panjang yang baru

      With Bucket

      • Pilih button Select
      • Pilih salah satu garis padacanvas
      • Pilih button Bucket
      • Gunakan slider untuk mengubah ukuran garis

  3. Mengubah ukuran sisi persegi

      With Move Point

      • Pilih button Select
      • Pilih salah satu persegi padacanvas
      • Pilih button Move Point
      • Klik pada salah satu titik sudut
      • Hover untuk menyesuaikan panjang sisi yang diinginkan
      • Klik pada canvas untuk mengubah posisi titik dan menghasilkan persegi dengan panjang sisi yang baru

      With Bucket

      • Pilih button Select
      • Pilih salah satu persegi padacanvas
      • Pilih button Bucket
      • Gunakan slider untuk mengubah ukuran persegi

  4. Mengubah ukuran sisi persegi panjang

      With Move Point

      • Pilih button Select
      • Pilih salah satu persegi panjang padacanvas
      • Pilih button Move Point
      • Klik pada salah satu titik sudut
      • Hover untuk menyesuaikan panjang sisi panjang/lebar yang diinginkan
      • Klik pada canvas untuk mengubah posisi titik dan menghasilkan persegi panjang dengan panjang sisi yang baru

      With Bucket

      • Pilih button Select
      • Pilih salah satu persegi panjang padacanvas
      • Pilih button Bucket
      • Gunakan slider untuk mengubah ukuran persegi panjang

  5. Menambah titik pada polygon

    • Pilih button Select
    • Pilih salah satu polygon padacanvas
    • Pilih button Add Point
    • Klik titik sudut baru yang diinginkan

  6. Menghapus titik pada polygon

    • Pilih button Select
    • Pilih salah satu polygon padacanvas
    • Pilih button Remove Point
    • Klik titik sudut yang ingin dihapus

  7. Melakukan rotasi

    • Pilih button Select
    • Pilih salah satu shape padacanvas
    • Pilih button Rotate
    • Hover pada canvas untuk menyesuaikan rotasi yang diinginkan
    • Klik pada canvas untuk mengakhiri rotasi

  8. Melakukan translasi

    • Pilih button Select
    • Pilih salah satu shape padacanvas
    • Pilih button Translate
    • Hover pada canvas untuk menyesuaikan titik akhir yang diinginkan
    • Klik pada canvas untuk mengakhiri translasi

  9. Melakukan dilatasi

    • Pilih button Select
    • Pilih salah satu shape padacanvas
    • Pilih button Bucket
    • Gerakkan slider untuk menyesuaikan ukuran yang diinginkan

  10. Menggerakkan titik sudut

    • Pilih button Select
    • Pilih salah satu shape padacanvas
    • Pilih button Move Point
    • Klik pada salah satu titik sudut
    • Hover pada canvas untuk menyesuaikan titik akhir yang diinginkan
    • Klik pada canvas untuk mengakhiri pergerakkan titik sudut

  11. Mengubah warna

    1. Mengubah warna sebelum menggambar

      • Pilih Color Picker
      • Pilih salah satu warna
      • Buat shape yang diinginkan

    2. Mengubah warna titik setelah menggambar

      • Pilih button Select
      • Pilih salah satu shape padacanvas
      • Pilih Color Picker
      • Pilih salah satu warna
      • Klik pada salah satu titik shape tersebut untuk mengubah warna

  12. Menghapus shape

    • Pilih button Select
    • Pilih salah satu shape padacanvas
    • Pilih button Delete

  13. Menghapus semua shape

    • Pilih button Clear Canvas

How to Save and Load

  1. Save Project

    • Pilih button Save Model
    • file baru dengan nama"model.json" akan tersimpan

  2. Load Project

    • Pilih button Choose File
    • Pilih file yang diinginkan dari lokal
    • Akan di-load file yang dipilih
